What did you do this past week?

This past week was one full of things to do. Between 3 projects and some writing to do I was short on time. I had started the project for this class a bit early. Luckily, my other class, also had a similar project and I was able to use my short Python program to produce acceptance tests so that I did not have to rewrite the test script. However, once I got rolling all went well.
